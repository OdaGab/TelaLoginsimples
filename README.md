## App de Tela de Login Simples em KotlinDescrição:

Este é um projeto de aplicativo Android básico que demonstra a criação de uma tela de login. 
A interface do usuário foi construída com ConstraintLayout, permitindo um layout flexível e responsivo que se adapta a diferentes tamanhos de tela.
O projeto também faz uso de Kotlin Extensions para simplificar o código no MainActivity.kt. 
As funções de extensão são usadas para:
•Exibir mensagens Toast de forma concisa.
•Configurar onClickListeners para botões de maneira mais limpa.
•Obter o texto de EditTexts com uma única chamada de função.
A lógica de autenticação é simples e valida um usuário e senha pré-definidos diretamente no código. 
Este projeto é um ótimo exemplo de como usar ConstraintLayout e Kotlin Extensions para criar um aplicativo Android moderno e eficiente.
